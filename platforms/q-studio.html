04
Architecture Patterns
Next.js App Router API
Route handlers as serverless functions. Server components for SEO pages. Client components for interactive booking flows.
Firebase + JWT Dual Auth
Firebase Admin SDK verifies ID tokens server-side. Google Sign-In, phone auth, and email OTP login paths. AuthContext wraps the app.
Zod Schema Validation
All API inputs validated with Zod schemas. Shared between react-hook-form resolvers and server-side route handlers.
Paise Currency Model
All monetary values stored in paise (integer). Display conversion at UI layer only. Prevents floating-point rounding errors.
Campaign-Driven Pricing
PricingCampaign model with date ranges. Active campaigns override base prices. Priority field resolves overlapping campaigns.
QR-Based Check-In
Unique qrToken per booking/pass. Physical desk QR codes for claim. Crypto-signed tokens prevent forgery.
Nonce-Based CSP
Middleware generates per-request nonce. Strict Content-Security-Policy with nonce and strict-dynamic in production.
Cron-Driven Lifecycle
14 Vercel cron jobs manage booking completion, desk release, pass expiry, reminders, abandoned cart recovery, and re-engagement.
