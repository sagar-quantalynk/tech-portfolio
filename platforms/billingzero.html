06
Architecture Patterns
Multi-Tenant Scoping
All queries scoped by vendor_id / parent_id. Vendors see only their own clients, contracts, and invoices.
Stripe Connect Platform
Platform account orchestrates connected accounts per vendor. Payment splits, fee calculations via StripeFeeCalculationService.
Encrypted Public Tokens
Public contract/invoice links use AES-256 encrypted tokens via EncrypDescrptService. No auth required for client-facing pages.
Service Manager Pattern
ContractServiceManager orchestrates 8+ sub-services. StripeServiceFactory for payment service instantiation.
DTO Value Objects
8 Data Transfer Objects: DepositResult, PaymentIntentData, TerminationResult, WebhookPaymentTotals, and more.
Webhook Idempotency
StripeWebhookEvent model logs all events. WebhookTransactionHelper prevents duplicate payment processing.
Scenario-Based Testing
288 test scenarios with MODULE-TYPE-NNN IDs. Coverage map syncing, gap detection, and staged-file coverage checks.
Tiered Rate Limiting
4 tiers: auth (10/min), public (30/min), api (60/min), webhook (1000/min). Plus upload-specific throttle.
