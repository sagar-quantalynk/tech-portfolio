3Outcome

05
Architecture Patterns
Multi-Tenant SaaS
User-scoped data isolation. Every query filters by userId. Plan-based feature gates limit access to premium features.
Global Tax Engine
Pluggable country tax calculators (IN/US/UK/UAE/CA/AU). TaxSystemType enum drives calculator selection. JSON taxBreakdown on invoices.
Dual Payment Gateway
Stripe Connect + Razorpay Route for seller payouts. Platform application fee in basis points. Unified webhook handling for both providers.
Multi-Channel Messaging
Unified Message model across WhatsApp, SMS, and Email channels. Template-based delivery with delivery/read tracking. SMS fallback.
Repository Pattern
Dedicated repository files abstract Prisma queries. Zod validation schemas per entity. Clean separation from route handlers.
Multi-Business Profiles
Single user can manage multiple business entities with different GSTINs, addresses, and bank accounts. Default profile switching.
Token-Based Portals
Client portal and CA access use unique access tokens. No account required for recipients. Expiry and permission-based controls.
Subscription Billing
Freemium with trial. Plan/Subscription/BillingEvent/UsageRecord models. Idempotent webhook processing. Usage-based limits enforcement.
